## Accuracy Rules

- Use dynamic verbs only where the public artifact supports them. Good defaults are **Engineered**, **Architected**, **Modeled**, **Implemented**, **Defined**, **Published**, and **Documented**.
- Reserve **Spearheaded**, **Monetized**, and similar ownership or outcome verbs for verified professional experience.
- Do not copy fictional scenario values into professional experience bullets.
- Label foundation projects as blueprints and commercial applications as prototypes.
- Keep a one-page resume to four projects. Link this portfolio for the complete seven-project system.
